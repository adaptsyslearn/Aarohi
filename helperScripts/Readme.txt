
####################  AAROHI: A compiler-based Predictor ##############################################

These are helper scripts used for processing files or computing statistics as needed. 

compute_mean_std_dev.sh and compute_meanstd_unopt.sh are essentially doing the same thing, only  Aarohi binaries (optimized versus unoptimized) and data file extensions (.log vs. .txt) are different. 

Helps automate a few things.
